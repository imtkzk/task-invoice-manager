# GitHubリポジトリ作成手順

## 1. GitHubでリポジトリを作成

1. https://github.com/new にアクセス
2. 以下の情報を入力:
   - Repository name: `task-invoice-manager`
   - Description: `タスク管理と請求書生成を統合した管理システム`
   - Public または Private を選択
   - **重要: "Initialize this repository with a README" のチェックを外す**
   - **重要: ".gitignore" と "license" は選択しない**
3. "Create repository" をクリック
